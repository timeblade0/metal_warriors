Controls:  
------------------  
All:  
  Esc = Exit  
  F2 = restart game  
  F4 = toggle fullscreen  
  Space = enter/exit mech  
  Enter = pause  

Nitro: (Basic Mech)  
  Left/Right = move  
  Up/Down = change aim  
  Z = Fly  
  A = shoot  

Drache: (UFO)  
  Arrow Keys = Move  
  S = shoot up  
  A = shoot left  
  X = shoot right  
  Z = shoot down  

Pilot: (No Mech)  
  Left/Right = Move  
  Z = Fly  
  A = Shoot
